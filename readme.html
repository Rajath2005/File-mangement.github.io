Rajath's File Management System

Overview

This project is a File Management System that allows users to upload, manage, and view files directly from the web interface. The system supports file upload with timestamps and displays a list of uploaded files, including their upload date and time. A confirmation message is shown after each successful upload.

Features

  • File Upload: Upload files easily through the web interface.
  • File List Display: Shows uploaded files with their names, upload date, and time.
  • Responsive Design: Professional and user-friendly layout.
  • File Management: Supports organizing and displaying uploaded files.
  • Password Protected Access: A secure login system with password fields styled in card-type structures.

Technologies Used

  • Frontend: HTML, CSS, JavaScript (for responsive and interactive UI)
  • Backend: Node.js (for handling file uploads and file listing)
  • File Storage: Files are stored on the server with timestamps.

Installation

Follow the steps below to set up the project:

  1. Clone the repository: git clone https://github.com/yourusername/filemanagement-system.git
  2. Navigate into the project directory: cd filemanagement-system
  3. Install the required dependencies: npm install
  4. Start the server: node app.js
  5. Open the website in your browser: http://localhost:3000

Usage

  • Upload files using the upload form on the homepage.
  • View the list of uploaded files, including their upload date and time.
  • The design offers a clean interface with password fields styled in card layouts.

Future Enhancements

  • File Preview: Enable users to preview certain file types.
  • Search/Filter Files: Add search functionality to filter files by name or upload date.
  • User Authentication: Add user login and authentication features.
